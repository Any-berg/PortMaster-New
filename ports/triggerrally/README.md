## Controls

| Button | Action |
|--|--| 
|DPAD| Steer|
|A| Accelerate|
|B| Brake|
|Y| Handbrake|
|X| Change Camera View|
|L1 | Map|
|L2| Toggle UI|
|R1 | Left Mouse|
|R2 | Recover Car|
|L3 | Pause|

## Compile

```shell
git clone https://github.com/Cebion/trigger-rally_pm.git
cd trigger-rally
cd src
make
```
