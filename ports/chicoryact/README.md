## Controls

| Button | Action |
|--|--| 
|Left-stick|Movement|
|Right-stick|Paintbrush|
|D-pad |Style Select|
|A|Interact / Confirm|
|B|Back / Skip|
|X|Map|
|L1|Brush Size|
|L2|Erase|
|R1|Brush Color|
|R2|Paint|
|R3|Zoom|
|Start|Menu|
|Select|Pause|
