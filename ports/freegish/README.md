## Controls

| Button | Action |
|--|--| 
|DPAD| Steer|
|A | Stick|
|B | Jump|
|X | Heavy|
|Y | Slide|
|L1/R1 | Y/N|
|L2 | C (Continue)|
|R2 | P (Practice)|


## Compile

```shell
git clone https://github.com/freegish/freegish.git
cd freegish
mkdir build && cd build
cmake ..
make
```
