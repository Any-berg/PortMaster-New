## Controls

| Button         | Action                            |
| -------------- | --------------------------------- |
| D-PAD/L-STICK  | Move                              |
| A              | Jump                              |
| X              | Dash                              |
| R1             | Skip cutscene                     |
| R2             | Hold to restart the current level |
| L2             | Hold to return to title screen    |
