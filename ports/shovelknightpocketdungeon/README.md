## Controls

| Button | Action |
|--|--| 
|D-pad / L-stick|Movement|
|A|Confirm / Items|
|B|Cancel / Ability|
|X|Ability|
|Y|Items|
|LB|Examine|
|RB|Turbo|
|Start|Pause|
|Start + Select|Quit Game|
