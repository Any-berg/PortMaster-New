# Controls

| Button       | Action                             |
| ------------ | ---------------------------------- |
| L-STICK      | Mouse                              |
| X            | Mouse-click / spit                 |
| D-PAD        | Movement                           |
| A            | Advance instructions               |
| B            | Jump                               |
| Y            | Restart level                      |
| R1/L1        | Page-up/page-down (in SELECT menu) |
| SELECT       | Escape/back                        |
| SELECT+START | Quit                               |
