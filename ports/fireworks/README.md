## Controls

| Button | Action |
|--|--| 
|DPAD / Left stick|Movement|
|A|Jump / Menu Confirm|
|B|Back in menu|
|X|Shoot|
|L1|Dash|
|L2 / R2 |Change weapon|
|Start|Pause menu|
