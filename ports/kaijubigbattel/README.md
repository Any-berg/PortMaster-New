Tested using the Windows version (Linux should work fine, and probably the Steam versions too).

## Controls

| Button | Action                      |
| -------| --------------------------- |
| D-PAD  | Walk around / control menus |
| A      | Action / proceed            |
| B      | Back                        |
| X      | Menu                        |


#Compile OHRRPGCE

```
Install FreeBASIC
git clone https://github.com/ohrrpgce/ohrrpgce.git
cd ohrrpgce
scons gfx=sdl2
```
