## Controls

| Button | Action |
|--|--| 
|DPAD / Left Stick|Movement|
|A|Fire shot 1|
|B|Fire shot 2|
|A+B|Fire shot 3|
|R1|Boost|
|L1|Use pickup|
|Start|Menu|
