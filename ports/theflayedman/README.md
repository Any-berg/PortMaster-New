# Controls

| Button        | Action            |
| ------------- | ----------------- |
| D-PAD/L-STICK | Mouse move        |
| A             | Left click        |
| B             | Right click       |
| X/Y           | Mouse slow        |
| L1            | Hint (if enabled) |
| SELECT        | Exit menu         |
