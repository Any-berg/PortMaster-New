OpenSupaplex

OpenSupaplex is a 1:1 reimplementation of the original game in C and SDL, with just a few changes to make it easy to use with game controllers, and also easy to port to any platform.


Developer: Sergio Padrino (https://github.com/sergiou87)
URL: https://github.com/sergiou87/open-supaplex


CONTROLS
========

- Exit: Hold start + select
- Up/Down/Left/Right: left joystick or directional pad
- Removing bases/infotrons next to Murphy: X button + direction
- Detonate red disk: hold X button
- Show/hide bottom game panel: Y button
- Show number of red disks: B button
- Change game speed: L and R buttons
- Exit/back: Select button
- Pause: Start button
