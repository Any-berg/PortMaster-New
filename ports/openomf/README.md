## Controls
Control mappings are set up for the 'Left keyboard' control scheme. To set this up, from the main menu navigate to Configuration -> Player 1 Input -> Select 'Left Keyboard'.

| Button | Action |
|--|--| 
|DPAD|Move|
|A|Punch|
|B|Kick|
|Select|Menu|
|Start|Select Item|
