## Controls

| Button     | Action                    |
| ---------- | ------------------------- |
| D-PAD      | Menu navigation           |
| START/B    | Menu confirm / proceed    |
| R-STICK    | Cast aim / mouse movement |
| R1/R2/R3   | Cast / mouse click        |
| L-STICK    | Movement                  |
| L1/L2/L3/A | Dash                      |
| SELECT     | Back / menu               |
