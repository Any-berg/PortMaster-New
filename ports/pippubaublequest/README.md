# Controls

| Button        | Action             |
| ------------- | ------------------ |
| L-STICK/D-PAD | Move               |
| A/B           | Jump / swim        |
| X/Y           | Run / breathe fire |
| START         | Pause              |
| SELECT        | Show bubbles       |
