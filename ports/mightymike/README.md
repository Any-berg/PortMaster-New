## Controls

| Button | Action |
|--|--| 
| X | Attack |
| B | Next Weapon |
| Y | Bunny Radar |


## Compile

```shell
git clone https://github.com/jorio/MightyMike.git
git submodule update --init --remote --force
mkdir build && cd build
cmake ..
make
```
