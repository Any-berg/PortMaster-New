## Notes
Thanks to [David Ashley](https://web.archive.org/web/20080820160645/http://www.linuxmotors.com/SDL_bomber/) for creating this game and making it available for free!

## Controls

| Button | Action |
|--|--| 
|DPAD| Drive|
|A| Lay Bomb|
|B| Timed Bomb|


## Compile

```shell
git clone https://github.com/Cebion/SDL-Bomber
cd SDL-Bomber
make
```
