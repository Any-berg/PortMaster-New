## Controls

| Button | Action |
|--|--| 
|D-pad|Move Between Orders|
|L-stick/ R-stick/ ABXY|Ingredients|
|L1|Coor Order/ Serve Prepared Order|
|L2|(Hold) More Ingredients|
|R1|Prepare Order/Serve Cooked Order|
|R2|Knife Cut (Soup)|
|Start|Pause Game|
