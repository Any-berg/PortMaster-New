## Controls

| Button | Action |
|--|--| 
|dpad|movement|
|left analog|movement|
|X button|punch|
|A button|regular jump|
|B button|crouch|
|Y button|fling jump|
|L1|reset level|
|L2|crouch|
|R2|fling jump|
