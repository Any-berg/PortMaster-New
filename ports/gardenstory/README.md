## Controls

| Button | Action |
|--|--| 
|D-pad/L-stick|Walk|
|A|Interact/Talk|
|B|Shield/Cancel|
|X|Inventory|
|Y|Attack|
|L|Map|
|ZL (hold)|Use Dew|
|ZR (click)|Roll|
|ZR (hold)|Dash/Run|
