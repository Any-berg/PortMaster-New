**This is game is local multiplayer only; there is no single player option**


## Controls

| Button       | Action           |
| ------------ | ---------------- |
| **Player 1** |                  |
| D-PAD        | Movement         |
| L-STICK-UP   | Jump alternative |
|              |                  |
| **Player 2** |                  |
| A/B/X/Y      | Movement         |
|              |                  |
| START        | Proceed          |
| BACK         | Pause/menu       |
