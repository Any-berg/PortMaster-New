## Controls

| Button | Action |
|--|--| 
|D-pad / Left stick|Navigate menu / Move |
|A|Select / Fire|
|B|Back / Jump|
|Y / L1|Ability 1|
|X / R1|Ability 2|
|Select|Pause menu / See map|
|Start + Select|Exit game|
