## Controls

| Button | Action |
|--|--| 
|DPAD/L-STICK|Movement|
|A|Confirm|
|B|Cancel/Back|
|X|Command Menu|
|START / Y|Confirm during character creation|
|START - DPAD DOWN|Interactive keyboard input for character naming|
