## Controls

| Button | Action |
|--|--| 
|D-pad/L-stick|Movement |
|A|Shoot|
|B|Roll|
|X|Saber|
|R1|Aim|
|L1|Next Gun|
|L2|Previous Gun|
|Select |Pause/Menu|
|Select |Reconfigure controls in Menu|
