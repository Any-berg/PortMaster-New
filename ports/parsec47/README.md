## Controls

| Button | Action |
|--|--| 
|DPAD| Move | 
|A| Shoot |
|B| Slow / Roll, Lock |
|x| Pause|
 
## Building

```
git clone https://github.com/Cebion/parsec47_pm.git
compile libbulletml in subfolder
sources/buildLinux.sh
```
