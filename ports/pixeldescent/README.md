## Controls

| Button  | Action      |
| ------- | ----------- |
| L-STICK | Mouse       |
| X       | Mouse click |
| D-PAD   | Move        |
| A       | Attack      |
| B       | Alt Attack  |
| Y       | Level up    |
| START   | Pause       |
| L1      | Cancel      |
| R1      | Auto-attack |
| R2      | Key markers |
