## Controls

| Button  | Action      |
| ------- | ----------- |
| D-PAD   | Movement    |
| R2      | Attack      |
| L2      | Alt Attack  |
| A       | Confirm     |
| B       | Cancel      |
| X       | Level up    |
| Y       | Key markers |
| START   | Pause       |
