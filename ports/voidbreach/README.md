## Controls

| Button  | Action            |
| ------- | ----------------- |
| D-PAD   | Navigate menus    |
| L-STICK | Mouse cursor      |
| A       | Mouse click       |
| Y       | Walk quick-select |
| B       | Inventory         |
| L1/R1   | Scroll icons      |
