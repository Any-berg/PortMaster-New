## Controls

| Button | Action |
|--|--| 
|DPAD| Steer|
|A | Bomb|
|B | Shoot|
|B(Hold)| Charge Shot|
|Y | Autofire|


## Compile

```shell
git clone https://github.com/Cebion/MagicalBroomExtreme.git
cd MagicalBroomExtreme
make
```
