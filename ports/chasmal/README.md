## Controls

| Button | Action |
|--|--| 
|DPAD / Left stick|Movement|
|Right stick|Aim|
|Right shoulder|Shoot|
|Left shoulder|Dash|
|A|Interact|
|B|Use An Active Item|
|X|Inventory|
|Y|Switch Weapons|
|Start|Pause menu|
