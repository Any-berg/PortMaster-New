To restart a level, click the leftmost icon at the top-right corner of the screen with your mouse.


## Controls

| Button  | Action      |
| ------- | ----------- |
| D-PAD   | Movement    |
| B       | Jump        |
| L-STICK | Mouse move  |
| X       | Mouse click |
