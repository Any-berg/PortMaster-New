# Controls

| Button  | Action        |
| ------- | ------------- |
| L-STICK | Movement      |
| A       | Jump          |
| X       | Right hand    |
| B       | Left hand     |
| Y       | Interaction   |
| RB      | Bombs         |
| SELECT  | Switch camera |
