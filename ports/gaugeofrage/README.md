## Controls

| Button | Action |
|--|--| 
|L-Stick|Movement|
|R-Stick|Aim|
|A|Select|
|B|Heal|
|X|Interact|
|Y|Swap Weapons|
|L2|Melee|
|R1|Grenades|
|R2|Shoot|
|R3|Reload|
|Start|Pause/Menu|
