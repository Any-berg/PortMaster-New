## Controls

| Button  | Action             |
| ------- | ------------------ |
| D-PAD   | Player movement    |
| L-STICK | Mouse cursor       |
| A       | Mouse hold/release |
| B       | Jump               |
| Y       | Retrieve all keys  |
| SELECT  | Menu/pause         |
