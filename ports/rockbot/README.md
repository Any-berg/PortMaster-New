## Controls

| Button | Action |
|--|--| 
|D-pad/L-Stick/R-Stick|Move|
|Select|Escape/Quit|
|Start|Enter/Make selection|
|A|Jump|
|B|Attack|
|X|Dash|
|Y|Shield|
|L1|Jump|
|R1|Attack|
|L2|Previous Weapon|
|R2|Next Weapon|
|L3/R3|Dash|

</br>

## Compile
### Rockbot
```shell
sudo apt install -y qt5-qmake gcc-7 g++-7
git clone https://github.com/slayer366/rockbot
cd rockbot
make -j4
```

### RockDroid
```shell
sudo apt install -y qt5-qmake gcc-7 g++-7
git clone https://github.com/slayer366/rockdroid
cd rockdroid
make -j4
```
