# Controls

| Button  | Action       |
| ------- | ------------ |
| D-PAD   | Movement     |
| L-STICK | Mouse cursor |
| A       | Mouse click  |
| B       | Inventory    |
| START   | Menu         |
| L1      | Tutorial     |
