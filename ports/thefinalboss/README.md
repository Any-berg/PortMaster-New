## Controls

| Button        | Action             |
| ------------- | ------------------ |
| D-PAD/L-STICK | Movement           |
| Y, B, A       | Use skills Q, W, E |
| R1, R2        | Use skills 1, 2    |
| L1, L2, X     | Use skills 7, 8, 9 |
| START         | Proceed            |
| SELECT        | Exit               |
