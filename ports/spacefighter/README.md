## 
This game has a full-screen/windowed option. If you wish to use this feature, you must press all 4 buttons, abxy each & push left, two times on the dpad at the same time. 

## Controls

| Button | Action |
|--|--| 
|dpad|move|
|left analog|move|
|a|fire|
|b|fire|
|start|pause|
