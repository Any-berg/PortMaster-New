## 
This game runs in full-screen mode. If you wish to use windowed mode, you must press all 4 buttons (A,B,X,Y) plus right on the d-pad, twice. 

## Controls

| Button | Action |
|--|--| 
|dpad|move|
|left analog|move|
|a|fire|
|b|fire|
|start|pause|
