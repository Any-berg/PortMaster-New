## Controls

| Button  | Action              |
| ------- | ------------------- |
| L1      | Start 1 player game |
| L2      | Start 2 player game |
| L-STICK | Mouse move          |
| B       | Mouse click         |
| UP      | Blue player         |
| X       | Red player          |
