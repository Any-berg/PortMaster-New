## Controls

| Button          | Action             |
| --------------- | ------------------ |
| D-pad / L-Stick | Movement           |
| X               | Interview          |
| A               | Back / Cancel      |
| B               | Interact / Talk    |
| Y               | Verify             |
| Left Top        | Absolutely nothing |
| Right Top       | Demand             |
| Left Bottom     | Clown Robot        |
| Right Bottom    | Interact / Talk    |
| Start           | Menu               |
