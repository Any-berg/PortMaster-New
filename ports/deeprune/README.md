## Controls

| Button  | Action          |
| ------- | --------------- |
| L-STICK | Mouse           |
| X       | Mouse click     |
| D-PAD   | Move            |
| A       | Interact        |
| B       | Face left/right |
| Y       | Click-to-move   |
| R1      | Run/walk        |
| L1      | Reset room      |
| L2      | Kill self       |
| SELECT  | Menu            |
