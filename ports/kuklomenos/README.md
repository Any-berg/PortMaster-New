## Controls

| Button | Action |
|--|--| 
|DPAD| Move|
|ESC| Menu|
|Start| Space (Start Game)|
|A| Gun 1|
|B| Gun 2|
|Y| Gun 3|

## Compile ## 

```bash
git clone https://github.com/Cebion/Kuklomenos.git
cd Kuklomenos
./configure
# adjust prefix to . 
make
```
