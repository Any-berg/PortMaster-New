Ybelle e Ludar apareceram num 
misterioso castelo. Deverão recolher 
os objetos mágicos em cada sala para 
poder passar à sala seguinte.
 
Controlos:
 
Setas - Mover a personagem
Barra de espaço - Saltar / Passar diálogo
Return - Saltar diálogo inteiro
F1 (a jogar) - Mostrar ajuda
ESC - Ocultar ajuda
