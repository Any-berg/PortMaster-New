## Controls

| Button | Action |
|--|--| 
|DPAD / Left stick|Movement|
|Right stick|Aiming|
|A|Use|
|Y|Drop weapon|
|X|Switch weapon|
|L1|Minimap|
|L2|Weapon wheel|
|R1|Melee|
|R2|Shoot|
|R3|Use item|
|Start|Menu|
