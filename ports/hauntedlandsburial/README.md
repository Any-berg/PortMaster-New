## Controls

| Button | Action |
|--|--| 
|D-pad|Movement |
|Y|Special attack|
|X|Shoot (hold for charged attack)|
|A|Jump|
|B|Dodge roll|
|R1|Manual reload|
|L1|In game help/pause screen|
|Start|Enter|
|Select|Escape|
