## Controls

| Button | Action |
|--|--| 
|L-stick|Direction|
|A|Primary Weapon|
|X|Roll/Dodge|
|R2|Accelerate|
|L2|Boost|
|L1|Secondary Weapon/Special|
|Start|Pause|
