## Controls

| Button | Action |
|--|--| 
|DPAD| Move Screen|
|B | Pass / Attack|
|Y| Shoot / Attack |


## Compile

```shell
git clone https://git.code.sf.net/p/tuxfootball/code tuxfootball-code
cd tuxfootball-code/
mkdir build && cd build
cmake ..
make
```
