## Controls
| Button | Action |
|--|--| 
|D-pad|Move|
|A|Use Tool|
|B|Jump|
|Y|Attack|
|X|Discard|
|L1|Swap Weapon|
|L2|Sprint|
|R1|Swap Tool|
|R2|Sprint|
|Start|Pause|
|Select|Inventory|
|Start + Select|Exit game|
