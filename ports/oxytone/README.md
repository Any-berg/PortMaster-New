## Controls

| Button | Action |
|--|--| 
|Dpad|Movement|
|Left stick|Camera|
|A|Place / Confirm (menu)|
|B|Discard / Cancel (menu)|
|X|New run (pause menu)|
|Y|Quit (pause menu)|
|Start|Pause menu|
|Optional:|For 2 stick devices, delete SDL_swap_gpbutton.txt if you want to use right stick for camera.|
