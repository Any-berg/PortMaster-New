## Controls

| Button | Action |
|--|--| 
|Dpad|Movement|
|Left stick|Camera|
|A|Place / Confirm (menu)|
|B|Discard / Cancel (menu)|
|X|New run (pause menu)|
|Y|Quit (pause menu)|
|Start|Pause menu|
|Note:|On devices with 2 analog sticks devices, camera is mapped on the right stick.|
