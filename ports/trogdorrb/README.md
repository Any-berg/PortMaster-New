## Controls

| Button | Action |
|--|--| 
|Dpad and Joysticks|Move|
|Start|Start|
|A|Confirm|
|B|Back|
|Y|Change resolution|
|X|Change overlay|
|Select|Quit while paused|
|Start + Select|Force quit|


## Compile

```shell
git clone https://github.com/Mode8fx/Trogdor-Reburninated.git
cd Trogdor-Reburninated/Trogdor-Reburninated
mkdir build
cd build
cmake ../ -DLINUX=ON
make
```
