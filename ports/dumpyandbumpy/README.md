## Controls

| Button | Action                        |
| ------ | ----------------------------- |
| D-PAD  | Navigate                      |
| A      | Lift, throw / confirm         |
| B      | Exit puzzle / back            |
| X      | Attack, hold to charge attack |
| Y      | Restart puzzle                |
| START  | Open menu                     |
