## Controls

| Button | Action |
|--|--| 
|DPAD| Move Cursor|
|A | Switch Pieces|
|B | Flood|


## Compile

```shell
dget - u http://deb.debian.org/debian/pool/main/v/vodovod/vodovod_1.10-5.dsc
cd vodovod-1.10
make 
```
