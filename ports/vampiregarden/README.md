## Controls

| Button | Action                                      |
| ------ | ------------------------------------------- |
| D-PAD  | Movement                                    |
| A      | Oscillate while waiting for the sun to pass |
| B      | Undo                                        |
| R1     | Reset                                       |
| SELECT | Exit                                        |
