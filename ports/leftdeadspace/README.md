## Controls

| Button | Action |
|--|--| 
|dpad|move-left,right,forward,back|
|left stick|move-left,right,forward,back|
|right stick|move-left,right|
|L1|shoot|
|L2|reload|
|R1|shoot|
|R2|reload|
|Start|quit/restart|
