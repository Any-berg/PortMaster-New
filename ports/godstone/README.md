## Controls

| Button | Action |
|--|--| 
|D-pad|Move|
|L-stick|Cursor|
|A|Interact|
|B|Left Click|
|X|Consumable 1|
|Y|Consumable 2|
|L1|Dodge|
|L2|Inventory|
|R1/R2|Jump|
|Start|Menu|
|Back|Return|
