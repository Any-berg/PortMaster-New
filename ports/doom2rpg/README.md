## Controls

| Button | Action |
|--|--| 
|DPAD| Look Around|
|A| Interact/Shoot/Talk|
|B| Skip Turn|
|Y| Map|
|X| Inventory|
|L1/R1 | Strafe|
|L2/R2| Switch Weapons|
|Select| Menu|


## Compile

```shell
git clone https://github.com/Cebion/DoomIIRPG-RE_pm.git
cd DoomIIRPG-RE
mkdir build && cd build
cmake ..
make
```
