## Controls

| Button     | Action            |
|------------|-------------------|
| A          | Accelerate        |
| B          | Brake             |
| R1         | Shift Up          |
| L1         | Shift Down        |
| DPAD       | Steer             |
| Function+A | Toggle fullscreen |


## Compile

```shell
git clone https://github.com/ptitSeb/f1spirit.git
make
```
