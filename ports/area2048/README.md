|--|--| 
|DPAD| Move | 
|A| Shot |
|B| Wide Shot |
|x| Pause|
 
## Building

```
git clone https://github.com/Cebion/area2048_pm.git
compile libbulletml in subfolder
cd a2k_src
./buildLinux.sh
```
