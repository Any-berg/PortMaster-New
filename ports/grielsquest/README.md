## Controls

| Button | Action |
|--|--| 
|DPAD| Move|
|A | Select / Restart Level|
|L1 | Toggle Fullscreen|
|Select| Exit|


## Compile

```shell
git clone https://github.com/M-HT/griels-quest.git
cd griels-quest
src/comun.h // change path to #define DATA_PATH "data/"
make
```
