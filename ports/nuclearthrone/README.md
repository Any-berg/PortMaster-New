## Controls

| Button | Action |
|--|--| 
|Left-stick|Movement|
|Right-stick|Aim|
|A|Confirm|
|B|Back|
|X|Pick Up / Use|
|Y|Swap|
|L2|Active|
|R2|Fire|
|Start|Pause / Menu|
