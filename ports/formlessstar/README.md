## Controls

| Button | Action                       |
| ------ | ---------------------------- |
| D-PAD  | Walk around / control menus  |
| A      | Use / talk to NPCs / confirm |
| B      | Open and close the menu      |


## Compile OHRRPGCE 

```shell
Install FreeBASIC
git clone https://github.com/ohrrpgce/ohrrpgce.git
cd ohrrpgce
scons gfx=sdl2
```
