## Controls

| Button | Action |
|--|--| 
|D-pad/Left Stick|Movement|
|Select|Escape/Quit|
|Start/A|Enter/Select/Restart Level|
|B|Undo last move|
|X|Show Hint|
|Y|Playback Level Solution|
|L1|Previous Level|
|R1|Next Level|
|L2|Level Select Screen|
|R2|Select Theme @Main Menu|
|L3|Fast-Forward|
|Right Stick + R3|Mouse (not required)|
