Thanks to [Philippe Brochard](http://hocwp.free.fr/sdlzombies.html) for making this game and making it available for free.

## Controls

| Button | Action |
|--|--| 
|DPAD| Move|
|L2| Use Map (m)|
|L1| Toggle Fullscreen(f)|

## Compile

```shell
git clone https://github.com/Cebion/sdlzombies_pm.git
cd sdlzombies_pm
./configure
make
```
